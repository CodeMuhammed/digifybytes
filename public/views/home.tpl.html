Bulk Mailer Instructions

On your spread sheet the column for "roles" specifies what kind of certificates should be sent to the individual whose detail is on that row

For example : To send a Digital 101 online certificate to a user, that users row should look like this.

Merron | Jones | merronjones@gmail.com | Digital101-Online

The following are a list of all available roles:

  • Trainer-ZA

  • Trainer-AfricaWide

  • Strategy101-ZA

  • Strategy101-Online

  • Strategy101-AfricaWide

  • Digital101-Online

  • Digital101-ZA

  • Digital101-AfricaWide

  • DigitalStrategy101-Online

  • DigitalStrategy101-ZA

Template Generator Instructions

The roles column of the CSV file indicates the template to be used when sending a certificate to a participant. To create a certificate template, do the following

  1. Load the template view
  2. Create a template category by adding an image from dropbox
  3. Drag the placeholder text to where you want the user name to appear on the template
  4. Customize the font information like its size style font-face and its color to match the template theme
  5. Preview to see how your template will look like when you send certificates to participants
  6. Tweak by repeating steps 2-5 for maximum satisfaction
  7. Save your template

Now when you proceed to the manual view, the new category will be added to the dropdown
